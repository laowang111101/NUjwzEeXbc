# 社区买菜系统 java538

## 项目概述

社区买菜系统是基于Spring Boot和MyBatis框架开发的，旨在为社区居民提供便捷的线上买菜服务。该系统后端采用Spring Boot和MyBatis技术栈，确保了系统的高效稳定。

## 技术栈

- **后端**: Spring Boot + MyBatis
- **数据库**: MySQL 5.7
- **开发工具**: IntelliJ IDEA

## 功能模块

- 用户登录/注册
- 商品浏览与搜索
- 购物车管理
- 订单处理
- 用户管理（区分超级管理员与普通用户）

## 系统角色

- **超级管理员**: 管理员账户，拥有最高权限
- **普通用户**: 可进行商品购买等常规操作

## 运行环境

- **操作系统**: 兼容性良好，可在常见操作系统上运行
- **Java环境**: 推荐使用JDK 1.8及以上版本

## 部署步骤

1. 准备MySQL数据库环境，版本推荐5.7
2. 导入项目提供的数据库文件
3. 使用IntelliJ IDEA等开发工具导入项目源码
4. 配置Spring Boot相关参数
5. 直接运行Spring Boot项目
